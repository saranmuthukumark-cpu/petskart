
##  Features of PETSKART

### Multi-Role System

* Separate login for **Admin, Vendor, and Client**
* Role-based dashboards and access control
* Secure authentication using JWT

---

### Animal & Product Listings

* Add pets (dog, cat, bird, etc.)
* Add livestock (cow, buffalo, goat, etc.)
* Add accessories (food, cages, toys, dress)
* Detailed listings with:

  * Images
  * Price
  * Age
  * Description
  * Location
  * Seller details

---

### Admin Features

* Manage all users (block/unblock)
* Full CRUD operations:

  * Add / Update / Delete pets, livestock, and products
* Manage veterinary, grooming, and training services
* Approve or reject vendor listings
* Monitor all orders and transactions
* Dashboard analytics (users, sales, listings)

---

### Vendor Features

* Vendor registration and login
* Add and manage pet/livestock listings
* Upload images and product details
* Edit/Delete listings
* Manage inventory
* View buyer inquiries
* Track orders and sales

---

### Client Features

* Register and login
* Browse pets, livestock, and accessories
* Search and filter by:

  * Category
  * Price
  * Age
  * Nearby location
* View detailed product and seller information
* Add to cart and checkout
* Online payment integration
* Save delivery address
* Track orders

---

### Smart Features

* Location-based nearby filter
* Direct buyer–seller contact (call/chat)
* Dual system:

  * E-commerce (cart & payment)
  * Direct selling (contact seller)

---

### Pet Care Information System

* Provides complete care details for each pet
* Includes:

  * Feeding guidelines
  * Maintenance and daily care tips
  * Vaccination details and schedule
  * Health and hygiene guidance
* Helps users make informed decisions
* Promotes responsible pet ownership

---

### 🛒 E-Commerce System

* Add to cart functionality
* Checkout process
* Secure payment integration (Razorpay/Stripe)
* Order confirmation and tracking

---

###  Security Features

* JWT-based authentication
* Role-based authorization
* Secure API routes
* Input validation

---

### System Features

* Real-time data handling
* Scalable MERN architecture
* Responsive UI (mobile + desktop)
* API-based communication (frontend ↔ backend)

---

### Unique Features

* Hybrid platform (e-commerce + marketplace)
* Supports pets and livestock
* Multi-vendor ecosystem
* Location-based discovery
* Direct buyer-seller interaction
* Integrated pet care guidance system
