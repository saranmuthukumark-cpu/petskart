## PETS ZONE - Pets & Livestock E-Commerce Platform (MERN Stack)

Project Overview:

Pets Zone is a modern e-commerce web application designed for pet lovers to easily browse and purchase pet-related products online.

It is designed for buying and selling pets, livestock, and pet-related products with a multi-role authentication system.

Each user (Admin, Vendor, Client) logs in and accesses a separate dashboard based on their role.

## User Roles & Features

1. Admin (Control Panel)

Features:

    login with admin role
    
    Manage all users (block/unblock vendors & clients)

    Full CRUD operations:

    Add / Update / Delete pets & livestock

    Add Veterinary services and accessories (food, cages, dress, toys)

    Approve or reject vendor listings

    View all orders and transactions

    Dashboard analytics (sales, users, listings)


2. Vendor (Seller Dashboard)

Features:

    Register/Login as vendor

    Create listings for:

    Pets (dog, cat, bird)

    Livestock (cow, buffalo, goat, etc.)

Upload:

    Animal images

    Price
  
    Age

    Location

    Description

    Manage listings (Edit/Delete)

    View buyer inquiries / contact requests

    Track their sales/orders


3. Client (Buyer)

Features:

    Register/Login as client

    Browse animals and products

Filter by:

    Category (pet/livestock/accessories)

    Price

    Age

    Nearby location

View: 

    Animal details (price, age, images)
      
    Seller contact info

Buy:

    Small pets & accessories via cart system (like Flipkart)

    Add to cart → Checkout → Payment → Order confirmation

    Save delivery address
