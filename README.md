## PETS ZONE - Pets & Livestock E-Commerce Platform (MERN Stack)

Project Overview:

Pets Zone is a modern e-commerce web application designed for pet lovers to easily browse and purchase pet-related products online.

It is designed for buying and selling pets, livestock, and pet-related products with a multi-role authentication system.

Each user (Admin, Vendor, Client) logs in and accesses a separate dashboard based on their role.
